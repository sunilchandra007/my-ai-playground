## 1. **What is an Agent Framework?**

## 2. **Core Components**
- **Agent**: The entity that performs actions.
- **Environment**: The context in which the agent operates.
- **Sensors**: Mechanisms to perceive the environment.
- **Actuators**: Mechanisms to interact with the environment.
- **Policy**: The decision-making logic.
